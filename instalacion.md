# Instal Vuex
```sh
npm install vuex --save
```

## Validación de formularios
```sh
npm install vee-validate
```

## Instalación Firebase

```sh
npm install firebase vuefire
```

## Instalar Bootstrap

```sh
npm install bootstrap --save
```

## Instalar Yup

```sh
npm install yup --save
```
